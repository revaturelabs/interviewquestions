5. What is the output of the following code?

public class Main {
   public int i = 0;
   public static void main(String[] args) {
       Main m = new Main();
       Main.Nested mn = m. new Nested();
       mn.printVariable(10);   
   }
   class Nested{
       public int i = 2;
       void printVariable(int i)
       {
           System.out.print(i+" ");
           System.out.print(this.i+" ");
           System.out.print(Main.this.i+" ");
       }
   }   
}

    A. 0 2 10
    B. 10 2 0
    C. 0 10 2
    D. Compile-time error

Answer: B
