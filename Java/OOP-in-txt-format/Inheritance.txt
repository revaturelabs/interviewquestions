1. Harry potter's Invisibility cloak was passed down to him by his father James and James potter got it from his father Henry potter, this is a clear example for?

    A. Hybrid Inheritance
    B. Multilevel Inheritance
    C. Single Inheritance
    D. Multiple Inheritance

Answer: B

2. English language vocabulary is comprised of old germanic, French, and Latin words, this is an example for?

    A. Hybrid Inheritance
    B. Multilevel Inheritance
    C. Single Inheritance
    D. Multiple Inheritance

Answer: D

3. In java, a class can not extend more than one class.

    A. True
    B. False

Answer: A

4. In java, combining two or many types of inheritance is considered as _______ Inheritance.

    A. Hybrid Inheritance
    B. Multilevel Inheritance
    C. Single Inheritance
    D. Multiple Inheritance

Answer: A

5. A superclass is extended by a subclass and the subclass is extended by another subclass. This is called as _______.

    A. Hybrid Inheritance
    B. Multilevel Inheritance
    C. Single Inheritance
    D. Multiple Inheritance

Answer: B

6. The relation between a superclass which gets extended by a single subclass is called _______ Inheritance.

    A. Hybrid Inheritance
    B. Multilevel Inheritance
    C. Single Inheritance
    D. Multiple Inheritance

Answer: C

7. A child class inherits all the properties of parent class so the child class can be implicitly upcasted to parent. This is called as _______.

    A. Upcasting
    B. Downcasting
    C. Typecasting
    D. None of the above

Answer: A

8. A parent class may or may not have all the properties of Child class, so parent class can be ecplicily downcasted to Child class. This is called as _______.

    A. Upcasting
    B. Downcasting
    C. Typecasting
    D. None of the above

Answer: B

9. main() method is a static method and static methods can't be overriden.
    A. True
    B. False

Answer: A

10. A static method can not be overriden if a sub class has a method with the same name as the static method in super class, its considered as _______.

    A. Data Hiding
    B. Overloading
    C. Overriding
    D. None of the above

Answer: A
