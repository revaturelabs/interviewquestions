#InlineandBlockelements

1. The CSS property that is used to specify the set the difference between two lines of your content is

A. line-height property
B. min-height property
C. max-height property
D. None of the above

ANSWER: A 
Explanation: The CSS line height property defines the amount of space above and below inline elements.


2. How to set an inline-block in CSS properties ?

A. color
B. display
C. block
D. style

ANSWER: B
Explanation: The display: inline-block syntax is used to set an inline-block in CSS.
