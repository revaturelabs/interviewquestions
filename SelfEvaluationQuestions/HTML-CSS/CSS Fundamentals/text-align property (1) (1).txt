
1. The CSS property used to control the text alignment is

A. text-style
B. text-align
C. text-size
D. None of the above

ANSWER: B
Explanation: text-align property is used to control the text alignment in CSS.


2. Which property is used to control the text size in CSS ?


A. text-style
B. text-size
C. font-size
D. None of the above

ANSWER: C
Explanation: font-size is used to control of size of the font in HTML elements.


3. Which of the following is the correct syntax to change the alignment of all paragraph elements to left ?

A. p {align: left;}
B. p {text-align: left;}
C. p {text-align; left;}
D. p {align; left:}

ANSWER: B
Explanation: The syntax to make the alignment of all paragraph elements to left is: p {text-align: left;}.


4. Which is the correct syntax for refering the external style sheet ?

A. <stylesheet> example.css </stylesheet>
B. <link rel="stylesheet" href="example.css">
C. <link> rel="stylesheet" href="example.css"</link>
D. None of the above

ANSWER: B
Explanation: The syntax is to define the external style sheet are <link rel="stylesheet" href="example.css">


5. Which of the following property is used to underline the text ?

A. text-decoration: underline;
B. text-decoration;
C. text-decoration:
D. text-decoration; underline;

ANSWER: A


6. Which CSS property is used to make a text bold ?

A. font: bold
B. style: bold
C. font-weight: bold
D. weight: bold

ANSWER: C
Explanation: The property font-weight:bold is used to set the boldness of the text.


7. Which of the following CSS property is used to display the hyperlinks without any underline ?

A. a {text-decoration: none;}
B. a {text-decoration; none;}
C. a {decoration: underline;}
D. a {decoration: no-underline;}

ANSWER: A
Explanation: text-decoration: none; is used to remove the underline from the links in HTML.


8. The CSS property is used to add shadows to the text ?

A. text-overflow
B. text-decoration
C. text-shadow
D. text-stroke

ANSWER: C
Explanation: text-shadow property in CSS is used to add shadows to the text.


9. Which of the following does not represent a value for the CSS property font-variant?

A. inherit
B. large-caps
C. small-caps
D. normal

ANSWER: B 
Explanation: CSS font-variant property values may be normal and small caps. It is used to set a font variant of an element.


10. The overflow text that is hidden from the user is represented by which of the following CSS properties?

A. text-shadow
B. text-overflow
C. text-hidden
D. None of the above

ANSWER: B 
Explanation: text-overflow property specifies the representation of overflow text which is not visible to the user.


11. Which of the following CSS properties is used to create the text a stroke ?

A. text-decoration property
B. text-transform property
C. text-stroke property
D. None of the above

ANSWER: C 
Explanation: The text-stroke property defines the color and width of strokes for text characters.

13. The property in CSS is used to set the indentation of the first line in a block of a text is ______

14. Findout the correct syntax to give a line over text.

A. text-decoration: none
B. text-decoration: line-through
