 
1) In CSS "color:pink" is defined as a: 
 
A.    Selector
B.    Rule 
C.    Declaration   
D.    Value

Answer: D


2) How do you alter an element's text colour?

 A. color:
 B. test-color:
 C. text:color
 D. fgcolor:

Answer: A


3) In CSS "font:size" is defined as a: 

A. Selector
B. Rule
C. Property
D. Property-Name 

Answer: D


4) Which selectors are employed to define a collection of items

A. name
B. id

A. bgcolor: 
B. background-color:
