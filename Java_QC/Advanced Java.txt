21. What is SPA?
Answer: which only loads a page once, and then dynamically updates the page. Any interactions with the page or subsequent pages do not require a round trip to the server, which means the page is not reloaded.
22. What is Server-Side Rendering?
Answer: when a user makes a request to a webpage, the server prepares an HTML page by fetching the data and sends it to the user’s machine over the internet. The browser then construes the content and displays the page. This entire process of fetching data from the database, creating an HTML page and sending it to client happens in milliseconds.
23. What is Client-Side Rendering?
Answer: Which is about rendering content in the browser using JavaScript. In this approach, communication with server happens only for getting the run-time data. All logic, data fetching, templating and routing are handled on the client rather than the server

26. What is one way data binding
Answer: It is to bind the data from the component to the view (DOM) or from view to the component. One-way data binding is unidirectional. You can only bind the data from component to the view or from view to the component.
27. What is two way data binding
Answer: Which will help users to exchange data from the component to view and from view to the component. It will help users to establish communication bi-directionally.
